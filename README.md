# fuckySheepGame

羊了个羊光速过关脚本，node版

抓包拿到`cat-match.easygame2021.com`下的请求，请求头中寻找字段`t`，复制粘贴即为你的token.

### 脚本内容
- 支持每日话题通关刷写（已测试）
- 支持日常挑战通关刷写（已测试）
- 支持自定义通关时间（没测试）
- 更多功能探索中...
